How it's calculated
Hard Cost = Σ(per sqm items) × Area  ·  Soft Cost = Hard Cost × Σ(soft % items)  ·  Annual Cost = (Hard + Soft) × Phase% each year
Annual Drawdown = Annual Cost × LTC  ·  Construction Interest = rolled into loan balance (PIK)  ·  Peak Debt = cumulative drawn + accrued interest
Operation Start = last construction year + 1  ·  NOI = (Rent − Opex) × Area × 12, escalating each year  ·  Exit Value = final NOI (escalated) ÷ Exit Cap
Unlevered IRR = XIRR(−Land Y0, −construction costs each yr, NOI + Exit)  ·  Levered IRR = XIRR(−equity each yr, levered CF, Exit)
